Set tag

{\% set <identifier> %}
  content
{\% end %}

The {\% set %} tag can be used to set the value of a variable in the current scope to the value of the tag's contents. It's best described with an example.

Example

Given the template:

{% set someHtml %}Here is some HTML!{% end %} 
{\% set foo %}
  {{ someHtml }}
{\% end %}

This would set the value of the variable foo to the string "{{ someHtml }}". Later on in your template, you can then do stuff like:


  {\{ foo }}
  {\{ html.safe(foo) }}

And get:


  {{ someHtml }}
  {{ html.safe(someHtml) }}
